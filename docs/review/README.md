snapshot**. Their command output, file-line references, scores, and blocker
wording describe the repository at the time each report was written; they are
not a live status dashboard.

Use
[`docs/superpowers/plans/2026-07-10-post-review-execution.md`](../superpowers/plans/2026-07-10-post-review-execution.md)
and its **Post-review hardening addendum** for current remediation state and

freshness, native `check` dispatch wording, deploy-key handling, and the EVM
security regressions changed after this snapshot.

The remaining release, packaging, SBOM/provenance, SDK distribution, link
checking, observability, and broader formal-verification gaps remain actionable
until the current execution ledger closes them.
